# Setlist Notes

## User story

As a user, I want to add notes to a setlist (e.g., tempo, capo per song) so that I can capture practice or gig details.

## Acceptance criteria

- [ ] Notes field editable on setlist
- [ ] Notes displayed in setlist view and/or player mode
- [ ] notes persisted to Firestore
- [ ] Supports free-form text (tempo/capo comments, cues, etc.)

## Notes

Setlist has notes?: string. Used for tempo/capo comments and other cues.
