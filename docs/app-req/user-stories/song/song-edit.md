# Edit Song

## User story

As a user, I want to edit a song's ChordPro content and metadata so that I can keep my chord charts up to date.

## Acceptance criteria

- [ ] ChordPro editor for song content
- [ ] Metadata fields editable: title, artist, key, capo, tempo, tags
- [ ] Changes persisted to Firestore
- [ ] updatedAt timestamp updated on save
- [ ] Works offline (with sync when back online)

## Notes

ChordPro format for content. Metadata: title, artist?, key?, capo?, tempo?, tags?.
